Student Performance Predictor — Full MLOps-Style ML System

  • Serving Layer: Flask web UI for form-based predictions; inputs flow through the saved preprocessing pipeline and selected model.
  • Config & Logging: Centralized config.yaml, structured logging, organized artifacts, and modular structure.
  • Skills Demonstrated: Data engineering, feature pipelines, model governance, exception handling, logging, reproducible packaging, and UI delivery.
